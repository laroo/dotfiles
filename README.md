# My Dotfiles

Moved away from Ansible as the installation itself required a lot of dependencies. Currently just use `bash` and `stow`

## Setup

### Ubuntu

- Run setup script

  ```bash
  git clone https://github.com/laroo/dotfiles.git ~/dotfiles
  ```

- Run setup script

  ```bash
  cd ~/dotfiles
  ./setup.sh
  ```
